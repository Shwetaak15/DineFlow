# 🍽️ DineFlow

### Smart Hostel Dining Management System

DineFlow is a full-stack MERN application designed to modernize hostel dining management through a centralized digital platform. It enables students to view daily menus, submit complaints, provide feedback, and track their requests, while allowing administrators to efficiently manage hostel dining operations through a secure role-based dashboard.

## ✨ Features

### 👨‍🎓 Student Portal

- Secure Login & Registration
- JWT Authentication
- View Daily Mess Menu
- Raise Complaints
- Track Complaint Status
- Submit Food Reviews
- View Personal Activity
- Update Profile

### 👨‍💼 Admin Portal

- Dashboard with Key Statistics
- Manage Weekly Menu
- Manage Students
- Review & Resolve Complaints
- Manage Student Reviews
